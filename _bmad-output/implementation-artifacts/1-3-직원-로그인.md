# Story 1.3: 직원 로그인

Status: done

<!-- Note: Validation is optional. Run validate-create-story for quality check before dev-story. -->

## Story

As a 병원 직원,
I want 아이디·비밀번호로 로그인하기를,
so that 내 계정으로 앱에 들어갈 수 있다.

## Acceptance Criteria

epics.md 원문 + architecture(토큰 인증)·DESIGN/EXPERIENCE에서 도출한 검증 가능한 항목입니다.

1. **(원문)** 등록된 직원 계정이 있을 때, 로그인 화면에서 **올바른 정보를 입력하면 홈 대시보드(`/`)로 이동**한다.
2. **(원문)** **잘못된 정보면 "로그인 실패" 안내**가 뜬다(로그인 화면에 머무름).
3. 백엔드 `POST /api/auth/login`(폼 형식: `username`, `password`) → 성공 시 **JWT 액세스 토큰** 반환(`{"access_token": "...", "token_type": "bearer"}`), 실패(없는 계정/틀린 비번) 시 **401**.
4. **비밀번호는 평문으로 저장하지 않는다** — 해시(pwdlib + bcrypt)로 저장·검증. 토큰은 PyJWT(HS256), 서명 비밀키는 코드가 아니라 **`.env`(`JWT_SECRET_KEY`)**.
5. **직원(Staff) 테이블**이 추가되고, 로그인 테스트용 **시드 계정**이 들어가 있다.
6. 로그인 성공 시 프론트가 토큰을 저장(localStorage)하고 홈으로 이동한다. (저장된 토큰으로 보호된 페이지 접근/로그아웃은 **Story 1.4**)
7. 로그인 화면은 DESIGN 다크 모드 토큰을 따르고, 접근성(터치 영역 44px 이상, 색만으로 정보전달 금지)을 지킨다.

### 권장 라이브러리 (2026-06 — FastAPI 공식 추천 최신 조합)
- **PyJWT** — JWT(토큰) 생성/검증. (구형 python-jose 대신 현재 권장)
- **pwdlib[bcrypt]** — 비밀번호 해시/검증. FastAPI 공식 튜토리얼이 passlib 대신 권장하는 최신 패키지. `PasswordHash.recommended()` 사용.
- **python-multipart** — `OAuth2PasswordRequestForm`(폼 로그인)을 받으려면 필수. 없으면 로그인 폼 파싱이 안 됨.
- 토큰 알고리즘 HS256, 비밀키는 `.env`. [Source: 웹 리서치 2026-06]
  - [FastAPI 공식: OAuth2 with Password, Bearer with JWT](https://fastapi.tiangolo.com/tutorial/security/oauth2-jwt/)

### 핵심 코드 가이드 (개발 시 참고 — 인증은 실수가 잦으니 정확히)
- **비밀번호 해시(pwdlib):**
  ```python
  from pwdlib import PasswordHash
  password_hash = PasswordHash.recommended()
  hashed = password_hash.hash("test1234")          # 저장용
  ok = password_hash.verify("test1234", hashed)    # 검증
  ```
- **토큰 생성(PyJWT):**
  ```python
  import jwt
  from datetime import datetime, timedelta, timezone
  def create_access_token(sub: str) -> str:
      exp = datetime.now(timezone.utc) + timedelta(minutes=settings.access_token_expire_minutes)
      return jwt.encode({"sub": sub, "exp": exp}, settings.jwt_secret_key, algorithm="HS256")
  ```
  ⚠️ `datetime.now(timezone.utc)` 사용(시간대 명시). `datetime.utcnow()`는 파이썬에서 폐기 경고.
- **로그인 엔드포인트:**
  ```python
  from fastapi.security import OAuth2PasswordRequestForm
  @app.post("/api/auth/login")
  def login(form: OAuth2PasswordRequestForm = Depends(), session: Session = Depends(get_session)):
      staff = session.exec(select(Staff).where(Staff.username == form.username)).first()
      if not staff or not password_hash.verify(form.password, staff.hashed_password):
          raise HTTPException(status_code=401, detail="아이디 또는 비밀번호가 올바르지 않습니다")
      return {"access_token": create_access_token(staff.username), "token_type": "bearer"}
  ```
- **프론트 로그인 호출(폼 형식 주의):**
  ```ts
  const body = new URLSearchParams({ username, password }); // form-urlencoded
  const res = await fetch(`${API_BASE}/api/auth/login`, {
    method: "POST",
    headers: { "Content-Type": "application/x-www-form-urlencoded" },
    body,
  });
  if (res.status === 401) { /* 로그인 실패 표시 */ }
  else if (res.ok) { const { access_token } = await res.json(); saveToken(access_token); router.push("/"); }
  ```
- **라우팅:** Next.js App Router는 `next/navigation`의 `useRouter().push("/")` 사용(`next/router` 아님).
