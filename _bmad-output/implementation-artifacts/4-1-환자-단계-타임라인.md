# Story 4.1: 환자 단계 타임라인

Status: done

## Story

As a 병원 직원,
I want 환자 통합 화면에서 환자가 지금 어느 단계(접수→진료→검사→수납)인지 점+선 타임라인 그래픽으로 보기를,
so that 진행 상황을 한눈에 파악하고 앞으로 남은 단계도 알 수 있다. (FR9, UX-DR4)

## Acceptance Criteria

1. **(원문)** 환자 통합 화면(`/patients/{id}`)에 **점+선 타임라인**이 표시된다 — 접수·진료·검사·수납 4개 단계를 가로로 배열하고, 각 단계 사이를 가로선으로 연결한다.
2. 단계 상태가 3가지로 시각적으로 구분된다:
   - **완료(done):** 초록(`success`) 채워진 원 + 체크 아이콘 + 초록 글자 + 초록 연결선
   - **현재(now):** 하늘(`accent-primary`) 채워진 원 + 주변에 옅은 글로우(ring) + 굵은 파란 글자
   - **예정(upcoming):** 회색(`bg-elevated`) 채워진 원 + 흐린 글자
3. 완료 단계의 **오른쪽 연결선**은 초록, 현재 단계의 **왼쪽 연결선**은 초록(완료에서 왔으니), **나머지**는 회색 — UX 목업 CSS(`.tnode.done`, `.tnode.now`)와 동일한 색 규칙.
4. 현재 단계가 무엇이든(`current_stage` 값) 타임라인이 올바르게 반영된다 — `STAGE_ORDER`(접수→진료→검사→수납) 순서 기준으로 완료·현재·예정을 계산한다. 알 수 없는 단계가 들어오면 첫 단계(접수)를 "현재"로 간주하고 화면이 깨지지 않는다.
5. **기존 텍스트 배지** (환자 헤더 카드 오른쪽의 `current_stage` 배지)를 **타임라인으로 교체**한다 — 헤더 카드 아래에 타임라인을 별도 카드/섹션으로 배치하고, 헤더에서 배지는 제거한다.
6. 타임라인은 **읽기 전용** 시각화 — `current_stage` 변경은 3.4의 `advance-stage` 엔드포인트가 담당하며, 이 스토리에서 단계를 바꾸는 버튼/인터랙션은 만들지 않는다.
7. **백엔드 변경 없음** — `current_stage`는 이미 `GET /api/patients/{id}` 묶음에 포함됨(2.1 설계). 기존 배너·카드·처방폼·체크리스트·WS 동작을 깨지 않는다.
8. UX-DR8 접근성: 단계 구분을 **색+아이콘+글자 3중**으로 표현한다(색만으로 정보 전달 금지).

> **범위 밖(하지 말 것):** 단계를 바꾸는 UI/버튼 = 3.4 `ProcedureChecklist`의 "다음 단계로"가 이미 함(이 스토리에서 추가 금지). / 환자 흐름판(전체 환자 현황 보드) = 4.3 범위. / 대기 초과 앰버 강조 = 4.4 범위. / 부서 간 자동 전달 = 4.2 범위. / 클릭 시 단계별 상세 팝업 = 후속.

### ⚠️ 이 스토리의 가장 중요한 설계 판단
- **완전히 프론트 전용.** `current_stage`는 이미 `GET /api/patients/{id}` 묶음에 있다(2.1에서 설계, 3.4에서 `advance-stage`로 변경). API 추가 없음.

- **STAGE_ORDER는 프론트에도 상수로 정의.** 백엔드와 순서가 다르면 타임라인이 틀린다 → 상수 값 동일하게 유지 (`["접수","진료","검사","수납"]`).
- **흰 테마 글로우 색.** DESIGN.md가 업데이트된 흰 테마(`accent-primary = #0050cb`)를 쓰므로, 글로우 색은 `rgba(0,80,203,0.2)` (목업의 하늘색 `rgba(56,189,248,0.25)` → 프로젝트 파랑으로 조정).
- **색+아이콘+글자 3중.** UX-DR8: 완료=초록+체크아이콘+초록글자 / 현재=파랑+radio아이콘+파란글자 / 예정=회색+아이콘없음+회색글자.

- ✅ **`current_stage` 값:** 3.4가 만든 `STAGE_ORDER = ["접수","진료","검사","수납"]`과 `advance-stage`로 관리됨. 타임라인은 이 값을 읽기만 함.

- ✅ **`Icon` 컴포넌트:** `fill={true}` prop으로 채워진 Material Symbols 아이콘 사용 가능. [Source: `frontend/src/components/Icon.tsx`]

### 확정 기술/패턴 가이드

```tsx
// frontend/src/components/StageTimeline.tsx
"use client";

import { Icon } from "@/components/Icon";

const STAGE_ORDER = ["접수", "진료", "검사", "수납"] as const;
type StageName = (typeof STAGE_ORDER)[number];
type StageStatus = "done" | "now" | "upcoming";

function getStatuses(currentStage: string): StageStatus[] {
  const idx = STAGE_ORDER.indexOf(currentStage as StageName);
  return STAGE_ORDER.map((_, i) => {
    if (idx === -1) return i === 0 ? "now" : "upcoming"; // 알 수 없는 단계 방어
    if (i < idx) return "done";
    if (i === idx) return "now";
    return "upcoming";
  });
}

export function StageTimeline({ currentStage }: { currentStage: string }) {
  const statuses = getStatuses(currentStage);

  return (
    <section
      aria-label="환자 단계 진행 현황"
      className="rounded-xl border border-border-subtle bg-bg-surface px-4 py-4"
    >
      <div className="flex items-start">
        {STAGE_ORDER.map((stage, i) => {
          const st = statuses[i];
          const isFirst = i === 0;
          const isLast = i === STAGE_ORDER.length - 1;
          // 왼쪽 선: 완료 또는 현재면 초록(이 단계까지 도달 완료)
          const leftGreen = st === "done" || st === "now";
          // 오른쪽 선: 완료인 경우만 초록(현재 이후로는 아직 미도달)
          const rightGreen = st === "done";

          return (
            <div
              key={stage}
              className="relative flex flex-1 flex-col items-center"
              aria-current={st === "now" ? "step" : undefined}
            >
              {/* 왼쪽 연결선 */}
              {!isFirst && (
                <div
                  className={`absolute top-[10px] right-1/2 h-[3px] w-1/2 ${leftGreen ? "bg-success" : "bg-bg-elevated"}`}
                />
              )}
              {/* 오른쪽 연결선 */}
              {!isLast && (
                <div
                  className={`absolute top-[10px] left-1/2 h-[3px] w-1/2 ${rightGreen ? "bg-success" : "bg-bg-elevated"}`}
                />
              )}

              {/* 점(dot) */}
              <div
                className={`relative z-10 mb-1.5 flex h-[22px] w-[22px] shrink-0 items-center justify-center rounded-full border-2 border-bg-base ${
                  st === "done"
                    ? "bg-success text-[#06281c]"
                    : st === "now"
                      ? "bg-accent-primary text-accent-on shadow-[0_0_0_4px_rgba(0,80,203,0.2)]"
                      : "bg-bg-elevated text-text-muted"
                }`}
              >
                {st === "done" && <Icon name="check" fill className="text-[10px]" />}
                {st === "now" && <Icon name="fiber_manual_record" fill className="text-[10px]" />}
              </div>

              {/* 단계 이름 */}
              <span
                className={`text-[11px] ${
                  st === "now"
                    ? "font-bold text-accent-primary"
                    : st === "done"
                      ? "font-medium text-success"
                      : "text-text-muted"
                }`}
              >
                {stage}
              </span>
            </div>
          );
        })}
      </div>
    </section>
  );
}
```

- ❌ 색만으로 완료/현재/예정 구분 → UX-DR8 위반. **아이콘+글자** 병행.
