### 이 스토리의 핵심 (한 줄)
**프론트엔드 화면의 "빈 집"을 짓는 작업**입니다. 가구(기능)는 아직 안 넣어요. Next.js라는 골조에 Tailwind라는 페인트(다크 모드 색), shadcn/ui라는 기성 가구 세트를 들여올 준비만 합니다. **백엔드(FastAPI)·데이터베이스(PostgreSQL)·로그인은 이 스토리 범위가 아니라 Story 1.2~1.3에서** 합니다 — 여기서 건드리지 마세요.

### 권장 버전 (2026-06 기준 최신 안정 조합)
- **Next.js 15+ (App Router)** + **React 19** — `create-next-app@latest`가 자동으로 최신 안정판 설치
- **Tailwind CSS v4** — ⚠️ **v3와 설정 방식이 완전히 다름.**
  - v4는 `tailwind.config.js`(JS 설정 파일)를 쓰지 않고, **CSS 파일 안에서 `@theme`로 설정**한다.
  - 콘텐츠 경로(content paths) 자동 감지 → `content: [...]` 설정 불필요.
  - `globals.css` 최상단은 `@import "tailwindcss";` 한 줄로 시작.
- **shadcn/ui** — 컴포넌트를 라이브러리로 설치하는 게 아니라 **내 프로젝트 폴더(`src/components/ui/`)로 코드를 복사**해 넣는 방식. 그래서 자유롭게 수정 가능. 접근성 좋은 부품. (★ 실제 설치된 base-nova 프리셋은 **Base UI** 기반 — `@base-ui/react` 사용. 예전 Radix 기반과 API가 다를 수 있으니 이후 스토리에서 참고)
- ※ 버전을 임의로 다운그레이드하지 말 것. `@latest`로 받은 조합을 그대로 사용.

### 📌 DESIGN.md 토큰 매핑 표 (globals.css `@theme`에 그대로 등록)
Tailwind v4에서는 `@theme` 안의 변수 이름이 그대로 유틸리티 클래스가 됩니다. 예: `--color-bg-base` → `bg-bg-base`, `text-bg-base` 등으로 사용.

```css
@import "tailwindcss";

@theme {
  /* 배경 */
  --color-bg-base: #0f172a;        /* 가장 어두운 바탕 (기본 배경) */
  --color-bg-surface: #1e293b;     /* 카드/패널 배경 */
  --color-bg-elevated: #334155;    /* 떠 있는 요소(모달 등) */
  --color-border-subtle: #334155;  /* 옅은 경계선 */

  /* 글자 */
  --color-text-primary: #f1f5f9;   /* 주요 글자 */
  --color-text-secondary: #94a3b8; /* 보조 글자 */
  --color-text-muted: #64748b;     /* 더 흐린 글자 */

  /* 포인트 (하늘색) */
  --color-accent-primary: #38bdf8;
  --color-accent-hover: #0ea5e9;
  --color-accent-on: #0f172a;      /* 포인트색 위 글자 */

  /* 상태색 */
  --color-danger: #f87171;         /* 위험/경고(빨강) */
  --color-danger-bg: #422006;      /* 경고 배경 */
  --color-warning: #fbbf24;        /* 주의(앰버) */
  --color-success: #34d399;        /* 완료/정상(초록) */
  --color-info: #38bdf8;           /* 정보(하늘) */

  /* 글꼴 */
  --font-base: 'Pretendard', 'Malgun Gothic', -apple-system, sans-serif;

  /* 글자 크기 */
  --text-display: 28px;
  --text-title: 20px;
  --text-body: 15px;
  --text-small: 13px;
  --text-caption: 12px;

  /* 둥근 모서리 */
  --radius-sm: 8px;
  --radius-md: 12px;
  --radius-lg: 16px;
  --radius-full: 9999px;

  /* 여백 (4의 배수) */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --spacing-xl: 32px;
}

body {
  background: var(--color-bg-base);
  color: var(--color-text-primary);
  font-family: var(--font-base);
}
```
- [Source: DESIGN.md#frontmatter(colors/typography/rounded/spacing)]
- ⚠️ shadcn init이 `globals.css`에 자체 CSS 변수(`--background`, `--foreground` 등)와 `:root`/`.dark` 블록을 추가할 수 있음. **DESIGN.md 색이 최종 적용되도록** 우리 `@theme` 값이 우선되게 정리할 것.
